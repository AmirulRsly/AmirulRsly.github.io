Project Description

My Degree Final Year Project titled "Intelligent Enemy in Turn-Based Battle System To Learn Kanamoji ". It is an educational game about learning japanese alphabet in a turn based combat system. In this project I have created a unique gameplay element which is a combination of turn based combat system and quiz game as the learning environment for the player. Using my previous knowledge about unity and c# programming I have developed an intelligent AI for the enemy behaviour by implementing finite state machine and fuzzy logic. This enables the AI to learn Kanamoji alongside the player to act as the competitor for the player on whoever gets to answer the quiz correctly and quickly.

At the time of writing this project is still in development. Although a demo of this project is available to download in the link provided above.
